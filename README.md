# AI Chat Hub

**AI Chat Hub** – это десктопное приложение на базе Electron, объединяющее в одном окне несколько популярных AI-чатов:

- 🧠 **DeepSeek**
- 🔵 **AliceAI**
- 🦆 **Duck AI**
- 💜 **GigaChat**
- 🐉 **Qwen AI**

## Возможности

- Быстрое переключение между сервисами через боковую панель.
- Поиск по текущей странице (`Ctrl+F` или кнопки в панели).
- Сворачиваемая боковая панель – остаются только иконки.
- Вход через email/пароль работает прямо внутри приложения (сессии сохраняются).
- Cookies и данные авторизации хранятся в отдельной папке пользователя.

## Установка

1. Скачайте установщик `AI Chat Hub Setup 1.0.0.exe` из раздела [Releases](https://github.com/mntrofficial/ai-chat-hub/releases).
2. Запустите установку и следуйте инструкциям.
3. После установки запустите приложение через ярлык на рабочем столе или в меню «Пуск».
